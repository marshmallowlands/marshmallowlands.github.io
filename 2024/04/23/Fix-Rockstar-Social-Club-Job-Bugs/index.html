电脑端

此脚本需要先安装油猴插件才能继续使用。其他浏览器，例如 360 浏览器，请自行测试是否有效。 如果你的浏览器已经安装了油猴插件的话，那么请直接安装脚本。

如果你用的是 Chrome 浏览器、Windows 系统自带的 Edge 浏览器，或者是 Firefox 火狐浏览器，
你需要先安装油猴插件，下面是安装链接：

适用于 Edge 浏览器的油猴（暴力猴）：
https://microsoftedge.microsoft.com/addons/detail/%E6%9A%B4%E5%8A%9B%E7%8C%B4/eeagobfjdenkkddmbclomhiblgggliao

适用于 Chrome 浏览器的油猴（在中国大陆的用户可能需要连接代理才能访问）：
https://chrome.google.com/webstore/detail/jinjaccalgkegednnccohejagnlnfdag

适用于 Firefox 火狐浏览器：
https://addons.mozilla.org/zh-CN/firefox/addon/violentmonkey

–

在安装完成后，点击下面的网址，以继续安装过程
https://greasyfork.org/zh-CN/scripts/493169-fix-rockstar-games-jobs-page

点击 “安装” 后，可能会弹出安装脚本的页面，点击安装
提示完成安装后，关闭安装页面和差事页面，然后再重新打开该页面，看看是否有效

手机端

手机端目前仅 Via 浏览器通过测试，其他浏览器请自行测试是否有效

直接在 Via 浏览器的设置中寻找 “脚本”，然后找到右上角的 ＋ 号，然后点击 “下载脚本”，然后粘贴这个链接：
https://update.greasyfork.org/scripts/493169/Fix%20rockstar%20games%20jobs%20page.user.js

验证效果

现在关闭页面，然后重新打开官网的差事页面，看看是否有效
